Reclamações sobre Notebook Acer Nitro V15 ANV15-51-58AZ

Leia as reclamações de outros usuários sobre o Acer Nitro V15.

Notebook esquenta muito rápido

Comprei este notebook para jogos, mas ele esquenta demais após apenas 30 minutos de uso. Muito decepcionado com o desempenho térmico.

Bateria não dura mais de 3 horas

A bateria descarrega muito rápido, mesmo quando estou apenas navegando na web. Esperava mais de um notebook novo.

Desempenho não é tão bom quanto anunciado

O desempenho é muito inferior ao esperado. Trava frequentemente e não consegue rodar jogos pesados como deveria.

Tela apresenta manchas escuras

Após alguns dias de uso, a tela começou a apresentar manchas escuras nos cantos. Isso é inaceitável para um produto tão caro.

Teclas começaram a soltar

As teclas começaram a soltar após apenas alguns meses de uso. Esperava uma construção mais robusta.

Wi-Fi desconecta frequentemente

O Wi-Fi do notebook desconecta várias vezes ao dia, tornando impossível trabalhar de forma consistente.

Áudio distorcido

O áudio fica distorcido quando aumento o volume. Não consigo usar para assistir filmes ou ouvir música.

Demora muito para carregar

O tempo de carregamento é muito longo. Demora horas para carregar completamente a bateria.

Portas USB pararam de funcionar

Após alguns dias de uso, as portas USB pararam de funcionar. Agora não consigo conectar meus dispositivos.

Ventoinha faz muito barulho

A ventoinha é extremamente barulhenta, mesmo quando o notebook está em repouso. Muito irritante.

Touchpad não responde corretamente

O touchpad é inconsistente e às vezes não responde ao toque. Difícil de usar sem um mouse externo.

Placa gráfica decepcionante

A placa gráfica não consegue rodar jogos modernos com boas configurações. Esperava mais de um notebook para jogos.

Muito pesado para transportar

O notebook é muito pesado para ser transportado com facilidade. Não é prático para viagens.

Incompatível com alguns softwares

Alguns softwares que uso regularmente não são compatíveis com este notebook. Muito frustrante.

BIOS travou

O BIOS travou várias vezes e tive que reiniciar o notebook. Isso nunca deveria acontecer.

Má ventilação

O sistema de ventilação não é eficiente e o notebook esquenta muito rápido.

Bluetooth desconecta frequentemente

A conexão Bluetooth é instável e desconecta frequentemente dos meus dispositivos.

Hardware defeituoso

Recebi o notebook com defeito no hardware. Tive que mandar para o conserto logo após a compra.
